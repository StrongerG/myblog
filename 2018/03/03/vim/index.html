vim使用入门

发表于 2018-03-03 | 分类于 Linux

主要快捷键

  • k j h l 分别对应上下左右
  • w 光标向前移动到下个单词首字母
  • e 光标向前移动到单词尾字母
  • b 光标向后移动到单词首字母
  • B 光标向后移动到字符串首
  • x删除字符
  • p粘贴 y复制 d剪切 ndd nyy
  • G 最后 nG回到第n行 gg回到文件首
  • u撤销 C+r 反撤销
  • ！python % 直接运行 python
  • sudo update-alternatives –config vim 在py和py2之间切换
  • NERDTree插件使用 :NERDTree进入 上下键选择 ctr+w+h光标到树形目录 ctr+w+l光标回右侧编辑窗口
    • o 打开关闭文件或者目录
    • t 在标签页中打开 T 在后台标签页中打开
    • ! 执行此文件
    • p 到上层目录 P 到根目录
    • K 到第一个节点
    • J 到最后一个节点
    • u 打开上层目录
    • m 显示文件系统菜单（添加、删除、移动操作）
    • ? 帮助 q 关闭
