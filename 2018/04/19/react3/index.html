一、基本样式设置

React 组件最终会生成 HTML，所以可以使用给普通 HTML 设置 CSS 一样的方法来设置样式。
如果我们想给组件添加类名，为了避免命名冲突，React 中需要设置 className prop。
此外，也可以通过 style prop 来给组件设置行内样式， style prop 要为一个对象。

关于 CSS Modules 的基本使用总结如下：

  • CSS Modules 产生局部作用域的唯一方法，就是使用一个独一无二的class的名字。
  • 并且会使用对象来保存原 class 和混淆后 class 的对应关系。
  • CSS Modules 只会转换 class 名相关的样式， id 选择器、伪类和标签选择器等会原封不动地出现在编译后的 CSS 中
  • 可以使用 composes 来组合样式
  • 使用 react-css-modules 库，通过高阶组件的形式来避免重复输入 styles.**
